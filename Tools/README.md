# Tools Directory

This directory contains development tools and helper scripts for the Onyx Terminal project.

## Planned Tools

- **Code Generators**: Scripts for generating boilerplate code
- **Migration Tools**: Database or configuration migration utilities
- **Build Scripts**: Custom build automation scripts
- **Test Helpers**: Utilities for testing and mocking

## Usage

Tools in this directory are for development purposes only and are not included in the final app bundle.

## Adding New Tools

When adding new tools:
1. Document their purpose in this README
2. Include usage instructions
3. Add any required dependencies to the project
4. Ensure tools are excluded from production builds
